# 💼 Portfólio - Darlei Santos Santana

Meu portfólio profissional desenvolvido para apresentar minha trajetória, habilidades, projetos e formas de contato.

O objetivo deste projeto é centralizar minha experiência como desenvolvedor, destacando projetos voltados para desenvolvimento web, aplicações móveis e games.

---

## 🚀 Tecnologias

Este projeto foi desenvolvido utilizando:

- React

- JavaScript (ES6+)
- HTML5
- CSS3
- React Icons
- Git
- GitHub

## 📸 Funcionalidades

- Apresentação profissional
- Sobre mim

## 📂 Estrutura do Projeto

```
src/
├── components/
│   ├── Nav.jsx         # Barra de navegação
│   ├── Hero.jsx        # Seção inicial (bio em formato de código)
│   ├── About.jsx       # Seção "sobre.md"
│   ├── Projects.jsx    # Cards de projetos (estilo explorador de arquivos)
│   ├── Skills.jsx      # Lista de habilidades (skills.json)
│   ├── Contact.jsx     # Contato em formato de terminal (contato.sh)
│   └── Footer.jsx      # Rodapé
│
├── App.jsx             # Componente principal
├── App.css             # Estilos dos componentes
├── index.css           # Design Tokens (cores, fontes e espaçamentos)
└── main.jsx            # Ponto de entrada da aplicação
```

---

## ▶️ Rodar localmente

**Requisito:** Node.js 18+ (recomendado 20 ou 22 LTS).

```bash
npm install
npm run dev
```

Abra o endereço mostrado no terminal (geralmente http://localhost:5173).

---

## 📦 Build de produção

```bash
npm run build
```

Visualizar a versão de produção localmente:

```bash
npm run preview
```
